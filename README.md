Aplicação mobile em que o usuário logado pode salvar contatos, como numa agenda comum.

O desafio foi de implementar a interação com os recursos nativos de galeria e câmera. Para isso eu implementei a funcionalidade de o usuário poder escolher uma imagem de perfil no ato do registro ou mudá-la depois de logado, podendo escolher da galeria ou tirar uma foto da câmera. Logo após escolher a imagem, o usuário deverá fazer o crop da foto para um formato quadrado, este corte irá ser saslvo na galeria na pasta DCIM. Ao adicionar um contato, o usuário logado pode adicionar uma imagem para o seu contato também.

Esta aplicação faz uso do back4app que é uma plataforma low code que permite criar rapidamente API's para projetos, utilizando o MongoDB

### Futuras Features:

Para as futuras funcionalidades da aplicação, estão as seguintes ideias:

- Campo de e-mail no cadastro de usuário
- Verificação de e-mail
- Página de recuperação de senha
- Possibilidade de registrar um contato sem escolher imagem
- Deletar usuário
- Possibilidade de editar os contatos já salvos
